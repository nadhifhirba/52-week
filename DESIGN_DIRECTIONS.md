# 52-Week Design Directions — Awwwards-Caliber Identities

Each app gets a UNIQUE design language. No template reuse. No generic SaaS layouts.

## 1. jastip-mate — Personal Shopper Tools
**Identity:** Japanese Convenience Store Luxe
**Inspiration:** Muji, Donki, 7-Eleven Japan, Uniqlo UI
**Colors:** White #FAFAFA, Soft Red #E60012, Teal #00857C, Warm Gray #F0EDE8
**Typography:** Zen Kaku Gothic (display), Noto Sans JP (body)
**Style:** Clean, organized, bright. Product-grid layouts. Rounded boxes with subtle shadows. Organized chaos like a well-stocked konbini. Barcode-inspired decorative elements.
**Hero:** Product calculator front-and-center with live price preview. Receipt-style summary cards.
**Motion:** Quick snap-in animations like items being scanned at checkout.

## Anti-Slop Checklist (apply to all)
- [ ] NO Inter or Roboto as primary font
- [ ] NO purple gradients on white backgrounds
- [ ] NO centered hero + three-column cards
- [ ] NO generic SaaS icon grids
- [ ] YES distinctive display font for headings
- [ ] YES bold, intentional color palette
- [ ] YES asymmetric or unexpected layouts
- [ ] YES one signature animation per app
- [ ] YES real-data focus (show the product, not abstract illustrations)
